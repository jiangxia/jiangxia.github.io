CSS盒模型

基本概念

盒子模型包括：content、padding、margin、border

标准模型和IE模型的区别

  • 标准模型的宽高为content的宽高
  • IE模型的宽高包括border

CSS如何设置这两种模型

  • 标准模型：box-sizing：content-box
  • IE模型：box-sizing：border-box

JS如何设置/获取盒模型对应的宽高

BFC

最常见的常见就是解决边距重叠。

  • 定义：BFC就是用来格式化块级盒子，即块级格式化上下文，同样管理inline类型的盒子还有IFC
  • 原理（渲染规则）：
    • BFC元素的垂直方向边距会发生重叠
    • BFC的区域不会与浮动元素重叠（可清除浮动）
    • BFC是独立容器，不与外部元素相互影响
    • 计算BFC高度的时候，浮动元素参与计算
  • 如何创建BFC：
    • float不为none
    • position不为static/relative
    • display的值为inline-block、table-cell、table-caption
    • overflow的值不为visible
  • BFC的使用场景
    • BFC元素为float时，也会参与高度计算
