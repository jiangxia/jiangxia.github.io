## 技术栈

- **前端框架**: HTML5 + TailwindCSS 3.0+
- **动画库**: fullpage.js + CSS3 Animations
- **图表库**: ECharts 5
- **字体**: Google Fonts (Inter + Noto Sans SC)
- **图标**: Font Awesome 6.4.0
- **架构**: 模块化ES6+ JavaScript

## 本地开发

1. **启动开发服务器**:
   ```bash
   python3 -m http.server 8000
   ```

2. **访问地址**: http://localhost:8000

3. **快捷键操作**:
   - `→` `↓` `Space`: 下一页
   - `←` `↑` `Backspace`: 上一页  
   - `Home`: 第一页
   - `End`: 最后一页
   - `1-9`: 跳转到对应章节
   - `F`: 全屏模式
   - `T`: 显示目录
